# Практичне заняття №1

Проєкт виконано з дисципліни **«Високорівневі мови програмування та фреймворки»**.

## Тема

Основи Python та робота з даними.

## Варіант

Варіант №11.

## Завдання

У проєкті виконано три завдання:

1. Обчислення суми двох чисел.
2. Перевірка числа на простоту.
3. Створення класу `Calculator` для додавання, віднімання, множення та ділення.

## Технології

- Python 3
- Visual Studio Code
